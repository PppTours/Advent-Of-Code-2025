**Nouvel event : Défi P++ de l'Advent of Code**
Pour ceux qui ne connaissent pas, https://adventofcode.com/ est un site internet sur lequel est publié tout le ans un calendrier de l'avent de défis de programmation. Chaque jour, un nouveau puzzle (découpé en deux parties) est ouvert et l'objectif est tout simplement... de le résoudre. Rien de bien compliqué en général, le but est avant tout de s'amuser.
Cela dit, cette année, nous allons pimenter un peu tout cela...

**Défi : 1 jour = 1 langage**, celui que vous voulez, mais jamais deux fois le même.

Et puisque un peu de compétition ne fait jamais de mal, un classement sera mis en place. Finir une moitié de puzzle vous accordera un certain nombre de points selon la complexité du langage, voici le barème (il est totalement arbitraire mais il fallait bien trouver un moyen de récompenser l'audace) :
- Langages "très haut niveau" (python, javascript...) => 1pt
- Langages objet orientés "modernes" (Java, Rust...) => 2pts
- Langages "old school" (C, Pascal...) => 3pts
- Eso-langs (Brainfuck et co...) => 5pts ainsi que mon respect vaguement horrifié
En cas de doute sur l'appartenance d'un langage à une catégorie, posez la question.

**Pour participer :**
Tout d'abord il faudra communiquer votre pseodo GitHub sur le serveur P++, afin que nous puissions vous authoriser à contribuer directement sur ce dépot.
Une fois ceci fait, chaque puzzle devra être soumis dans le dossier "JourX" correspondant (créez le dossier s'il n'existe pas encore), avec un nom de fichier de la forme suivante : 
- _**"Pseodo_language[_PartieN].extension"**_, par exemple "Itreza2_C.c"

Cette année l'advent of code ne durera que 12 jours, mais vous aurez jusqu'au 24 décembre pour soumettre vos solutions, le leaderboard sera publié juste après.

On espère que l'idée vous plaira !
